# HTML Resume Page

## 📌 Overview
This is a single-page resume website built using only HTML. 
No CSS has been used as per assignment instructions.

## 🚀 Features
- Semantic HTML structure
- Header, About, Skills, Experience, Projects, Education, Contact sections
- Proper use of headings, lists, tables, and sections

## 🛠️ How to Run
1. Download the repository
2. Open `index.html` in any browser
