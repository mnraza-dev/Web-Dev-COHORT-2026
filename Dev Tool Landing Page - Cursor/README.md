# Cursor Landing Page Recreation

A high-fidelity recreation of the [Cursor.com](https://cursor.com) landing page, built as a desktop-first developer tool marketing site.

## Screenshots

*(Screenshots will be visible once the `index.html` is opened in a browser)*

## Sections Recreated

This project includes all 11 core sections of the Cursor landing page:
1.  **Top Navigation Bar**: Fixed header with logo, product links, and call-to-action buttons.
2.  **Hero Section**: High-impact headline, description, download buttons, and a product screenshot.
3.  **Trusted By**: Social proof section featuring a row of world-class company logos.
4.  **Feature Sections**: Three detailed blocks highlighting Cursor's core capabilities (Agents, Parallel Work, Autocomplete) with alternating layouts.
5.  **Feature Cards**: A grid highlighting key architectural benefits (Models, Codebase Indexing, Enterprise Security).
6.  **Testimonials**: Grid of quote cards from industry leaders.
7.  **Recent Highlights**: Blog-style grid showcasing research updates and customer stories.
8.  **Changelog**: A clean, chronological list of recent product updates.
9.  **Team / About**: Section highlighting Cursor's mission as an applied research team.
10. **Final CTA**: A concluding section to drive product downloads.
11. **Footer**: Detailed site navigation, company information, and social links.

## Design Details

- **Typography**: Primary font is **Inter** from Google Fonts, matching the clean, modern aesthetic of the original site.
- **Color Palette**:
  - **Backgrounds**: Pure Black (`#000000`) and Near-Black (`#0a0a0a`) for section differentiation.
  - **Text**: White for primary headings, slate-gray (`#a1a1a1`) for body text, and muted gray (`#888888`) for secondary details.
  - **Accents**: Subtle blue (`#4da6ff`) for links and highlight tags.
- **Visuals**:
  - Custom-generated images for the Hero screenshot and key feature visuals.
  - Subtle borders (`rgba(255, 255, 255, 0.08)`) and high-end shadowing for depth.
  - Transparent glassmorphism effects on the navigation bar.

## Constraints Followed
- **Pure HTML and CSS**: No JavaScript or TailwindCSS used.
- **Desktop-First**: Optimized for a 1200px+ viewing experience.
- **No Animations**: Focused on structural and visual accuracy as requested.
